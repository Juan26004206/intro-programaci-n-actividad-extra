# Actividad Extra – Introducción a la Programación

Este repositorio contiene la Actividad Extra de la Unidad III del curso *Introducción a la Programación de Computadoras*.

## Estructura del repositorio

- **main**: Contiene únicamente este archivo README.md y la licencia seleccionada.
- **actividades**: Incluye las carpetas organizadas por unidad (Unidad1, Unidad2, Unidad3) con los archivos correspondientes.
- **resúmenes**: Contiene los resúmenes de cada unidad, escritos con mis propias palabras.

## Licencia seleccionada

Elegí la licencia **MIT** porque es una licencia simple, abierta y flexible.  
Permite que cualquier persona pueda usar, modificar y compartir el código sin restricciones complicadas, lo cual es ideal para proyectos educativos.
